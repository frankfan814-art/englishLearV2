# 闲鱼自动发货与云激活运营手册

> 配套文档：`docs/商业模式与防盗版方案.md`（商业分析）、`cloud/README.md`（后端部署）
> 本文只讲日常运营：怎么自动发货、退款怎么锁、售后怎么处理。

## 整体流程

```
你：生成卡密 → 导入自动发货工具 → 闲鱼上架
买家：拍下付款 → 自动收到【下载链接 + 卡密】→ 安装 App → 输入卡密激活
你（退款时）：一条命令作废卡密 → 买家下次打开 App 联网即被降级回试用版
```

## 一次性准备（只做一次）

1. **部署云激活后端**：按 `cloud/README.md` 操作（约 20 分钟），并把环境 ID 和闲鱼商品链接填入 `src/config/license.ts` 后重新打包。
2. **配置卖家凭证**（本机一次）：

   ```bash
   # Git Bash
   export TCB_SECRET_ID=你的SecretId
   export TCB_SECRET_KEY=你的SecretKey
   export TCB_ENV_ID=你的环境ID
   ```

   密钥在腾讯云控制台「访问管理 → API密钥管理」创建。**不要发给任何人，不要提交进 git。**
3. **生成首批卡密**：

   ```bash
   node scripts/license_admin.js gen 50
   ```

   输出的列表一行一个卡密，整段复制备用（这就是你的"库存"，卖一个少一个）。
4. **固定 APK 下载链接**：沿用现有 CI（push main → 自动构建 → 蒲公英 `latest`），把蒲公英下载短链作为发货内容的一部分，链接永不过期、永远指向最新版。

## 自动发货方案

### 方案一：阿奇索（推荐，省心）

第三方虚拟商品自动发货工具，支持闲鱼，拍下后秒发。

1. 注册阿奇索，绑定闲鱼店铺（按其引导授权）。
2. 创建商品绑定：选择"**卡券/卡密模式**"（每个买家发不同的码），把上一步生成的 50 个卡密粘贴进库存。
3. 设置发货消息模板（见下文），库存低于阈值时它会提醒你补货——补货就是再跑一遍 `gen`。
4. 费用：按月订阅（以官网实时价格为准），先开一个月试水即可。

### 方案二：闲鱼自带自动发货（免费，有门槛）

1. 发布商品时选择**虚拟物品**类目（网盘/卡券/直充类），部分账号需先开通「鱼小铺」。
2. 在商品管理中配置卡密发货，粘贴卡密库存。
3. 局限：类目和账号有开通条件，虚拟商品曝光权重低于实物，且平台对"软件/App"类描述审核较敏感——文案里避免"永久""破解"等词。

### 发货消息模板（直接抄）

```
感谢支持！【单词朗读】安卓版

① 下载安装（复制到浏览器打开）：
https://你的蒲公英下载链接

② 你的专属激活码（每台手机限绑一台）：
{卡密}

③ 激活方法：打开 App → 首页点"激活完整版" → 输入激活码即可。

激活码请妥善保管，虚拟商品售出不退。使用问题回复本消息即可。
```

## 退款处理 SOP（核心）

1. 买家在闲鱼申请退款 → 你同意退款。
2. 立即执行：

   ```bash
   node scripts/license_admin.js revoke VCAB-XXXX-XXXX
   ```

3. 完成。买家下次打开 App 时启动校验会发现卡密已作废，自动降级回试用版（限前 200 词），已解锁内容立即收回。
4. 若买家一直不联网：离线宽限期 7 天后同样自动降级，联网校验通过前不会恢复。

> 建议商品详情写明"虚拟商品售出不退"，能挡掉大部分退款纠纷；真遇到平台介入判退的，按上面流程锁码即可，损失为零。

## 换机 / 售后 SOP

- **买家换手机**：让买家把新 App 里「设置 → 软件激活」页面的**设备码**发给你，核对购买记录后执行：

  ```bash
  node scripts/license_admin.js unbind VCAB-XXXX-XXXX
  ```

  买家在新手机输入原卡密即可重新激活（旧手机随后校验失败会自动降级）。
- **看销量 / 对账**：

  ```bash
  node scripts/license_admin.js list
  ```

- **补库存**：`node scripts/license_admin.js gen 50`，把新卡密追加进发货工具库存。

## 常见问题

- **买家说激活失败"已绑定其他设备"**：基本是家里第二台手机或转发给别人了，按换机流程处理或礼貌拒绝。
- **买家手机恢复出厂设置**：ANDROID_ID 会变，等同换机，unbind 后重新激活。
- **卡密输错提示不存在**：让买家复制粘贴（不要手打），卡密不含 0/O/1/I。
- **想给亲友免费用**：自己构建时建一个 `.env.local` 文件写 `VITE_LICENSE_DISABLED=true` 再打包，这个包全功能免激活——**千万保管好这个包，不要外传**。
